# Neural Network Fundamentals and Training Behavior Analysis

## Project Overview
This project demonstrates the implementation of a feed-forward neural network for customer churn prediction using TensorFlow and Keras.

## Tasks Completed
- Dataset understanding and exploration
- Data preprocessing
- Neural network model building
- Training and evaluation
- Hyperparameter experimentation
- Final reflection and analysis

## Technologies Used
- Python
- TensorFlow / Keras
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Model Performance
The neural network achieved high testing accuracy with effective classification performance on customer churn prediction.
